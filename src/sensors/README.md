# Sensors module


This module contains the sensor classes that are used in the project.

- `sensors.rs`- Aboslute magnetic encoder `as5027d` and `RLS Aksim2` and `Hall sensors` array
- `analog.rs` - Temperature sensors `NTC`
- `ltc4332.rs` - Communication with the LTC4332 chip which is used to communicate sensors from the `PVT` version
